## Sistem Pemesanan Barang Berbasis QR Code - ID

Mengembangkan aplikasi pemesanan barang berbasis web yang memanfaatkan teknologi QR Code untuk mempermudah proses pemesanan dan meningkatkan pengalaman pengguna. Pelanggan dapat melakukan pemindaian QR Code untuk langsung mengakses daftar produk, melakukan pemesanan, serta mengirimkan permintaan pembelian tanpa memerlukan proses manual.

Sistem ini dirancang untuk meningkatkan efisiensi operasional, mempercepat proses pemesanan, dan menyediakan pemantauan transaksi secara real-time. Selain itu, tersedia dashboard administrasi untuk mengelola produk, memantau pesanan, dan menghasilkan laporan operasional.

**Fitur Utama:**

* Pemesanan produk melalui QR Code.
* Tampilan responsif untuk desktop dan perangkat mobile.
* Manajemen pesanan secara real-time.
* Pengelolaan produk dan stok barang.
* Monitoring transaksi dan laporan penjualan.
* Dashboard administrasi yang mudah digunakan.

**Teknologi:**
PHP, CodeIgniter/Laravel, JavaScript, Bootstrap, MySQL, QR Code Integration, REST API.



## QR Code Self-Ordering Web Application - EN

Designed and developed a web-based self-ordering platform that allows customers to scan a QR Code and place orders directly from their smartphones. The solution eliminates manual ordering processes, enhances operational efficiency, and improves customer convenience.

The application includes menu management, order processing, real-time order status updates, and reporting features to support daily business operations. The system was built with a focus on responsiveness, scalability, and ease of use across multiple devices.

**Key Achievements:**

* Reduced manual order processing.
* Improved ordering speed and customer experience.
* Enabled real-time order tracking and management.
* Streamlined business operations through digital ordering.

**Technologies:**
PHP, CodeIgniter, MySQL, Bootstrap, JavaScript, QR Code Technology, REST API.
